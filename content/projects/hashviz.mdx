## Project Overview
**HashViz** is one of my very first websites, made to explain hash collisions whilst I was learning about data structures. 

### Key Features
- **Interactive Hash Map:** The user can make entries into a hash map and see both in text and visually how collisions are stored.

### Technologies Used
- **Frontend:** HTML 5, CSS3, Javascript
- **Deployment:** AWS EC2 Ubuntu instance running Apache (LAMPP)
