## Project Overview
**Luisa Brito's Portfolio** is a full-stack NextJS design portfolio. It allows Luisa to add new projects easily, specifying custom grids. The website was built according to Figma designs with a focus on simplicity, image optimisation, and accessibility.

### Key Features
- **User Interaction:** Luisa can upload new projects without further development, specifying custom grids and line breaks.

### Technologies Used
- **Frontend:** NextJS/React, Tailwind CSS, shadcnui
- **Backend:** NextJS
- **Deployment:** AWS EC2 Ubuntu instance running NGINX
