## Project Overview
**Zeldle** was my first React project. It helped me get started with the React library and state, but also with deployment, SSH, the Linux CLI, and NGINX. Solutions are stored in a Supbase SQL table with RLS.

### Key Features
- **The Game**: A Zelda themed wordle clone with dynamic database solutions.

### Technologies Used
- **Frontend:** React, CSS3, SQL (Supabase)
- **Backend:** Supabase SQL
- **Deployment:** AWS EC2 Ubuntu instance running NGINX
