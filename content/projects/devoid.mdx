## Project Overview
**DEVoid** is a full-stack React social media platform. I built it to better understand material UI, reusability, scalability, and social media platforms.

### Key Features
- **User Interaction:** Users can test without an accuont, sign up, post and upload images, like posts, and add / remove friends.

### Technologies Used
- **Frontend:** React, Tailwind CSS, shadcnui
- **Backend:** MongoDB, AWS
- **Deployment:** AWS EC2 Ubuntu instance running NGINX
